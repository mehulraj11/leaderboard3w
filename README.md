# 🏆 User Points Ranking System

A dynamic ranking application where users can select from **10 predefined users**, claim **random points**, and view **real-time rankings** based on total points.  

**Backend:** Node.js (Express)  
**Frontend:** React.js + Tailwind CSS  

---

## 🚀 Features

- **User Selection** – Choose one of ten available users.
- **Claim Random Points** – Assign random points to the selected user.
- **Dynamic Rankings** – Automatically updates rankings based on total points.
- **Real-Time Updates** – Points and rankings refresh instantly after actions.
- **Responsive UI** – Built with TailwindCSS for a clean and modern design.

---

## 🛠 Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** (Optional) MongoDB / In-Memory Storage
